# Spring Boot Generic WebClient Utils

A simple, reusable utility for making HTTP requests in Spring Boot using WebClient.

## Features

- Generic WebClient service
- Supports `GET`, `POST`, `PUT`, `PATCH`, and `DELETE`
- Easy to integrate and reuse in any Spring Boot project

## Usage

Inject the service and call the desired method

## Technology Stack

Java 21

Spring Boot 4.0.0

spring-boot-starter-webflux (for WebClient)

Maven (build and dependency management)
